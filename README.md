# ATS Resume Analyzer

An AI-powered resume analysis platform that evaluates resumes against job descriptions using Natural Language Processing (NLP). The system extracts skills, identifies missing keywords, calculates ATS scores, and provides recommendations to improve resume quality.

## Features

* Resume Parsing
* ATS Compatibility Scoring
* Job Description Matching
* Keyword Extraction
* Missing Skills Detection
* Resume Improvement Suggestions
* Skill Analysis Dashboard
* NLP-Based Text Processing

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Python
* Flask

### AI / NLP

* SpaCy
* Scikit-learn
* TF-IDF
* Natural Language Processing

## Workflow

1. Upload Resume
2. Upload Job Description
3. Extract Skills and Keywords
4. Compare Resume with Job Requirements
5. Generate ATS Score
6. Display Improvement Suggestions

## Installation

### Clone Repository

```bash
git clone https://github.com/dharshini0507/ats-resume-analyzer.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```
